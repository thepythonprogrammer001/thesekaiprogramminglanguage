# Learning to code in SEKAI

In SEKAI, You can make code only with variables with number or text values.

Example of a text value:

```
<token> = "<your text here>"
<token>
```

Example of a number value:

```
<token> = / + / - <value>
<token>
```

Have fun with your code!
